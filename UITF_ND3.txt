# 1. DEUTERATED AMMONIA (ND3) TARGET MATERIAL
#
# Molecular composition: ND3 = N14 + 3D

# Stoichiometric mass fractions:

#   N14 fraction = 14.003 / 20.045 ~ 0.6986
#   D fraction = 6.042 / 20.045 ~ 0.3014
#
# Bulk density: 0.5946 g/cm3 (locked to status-talk basket specs: 37 cc, 22 g)
# Mean excitation energy: 53.0 eV (typical for N/D compound)
###############################################################################

# Density specification (NIST values at 1 atm):
#
#   87 K irradiation (10 uA) — NORMAL GAS:

#             Ideal gas: rho = PM/(RT) = 5.601e-4 g/cm3; virial correction < 0.12%
#
#   2 K irradiation (1 uA) — COMPRESSED LIQUID He-II (superfluid):

#             isobaric table at P = 0.101325 MPa (1 atm), T = 2.0 K
#             Phase: He-II (below lambda point T_lambda = 2.1768 K at sat.)
#             Saturation pressure at 2 K ~ 3.13 kPa << 1 atm -> compressed liquid
#
#   *** SET DENSITY BELOW TO MATCH THE RUN TEMPERATURE BEFORE EXECUTING ***

# Raster dimensions: full width x height.

# FIX-7/FIX-9 (2026-06-12): symmetric end-compensation beamlets for axial NUI.

# arithmetic in parameter expressions.

# Reference beam current for MC-to-physical scaling
# FIX (2026-06-08): TOPAS only accepts base SI unit A (ampere). uA is not
# recognized. 1 uA expressed in amps = 1e-6 A; 10 uA = 1e-5 A.
#
# *** SET CURRENT BELOW TO MATCH THE RUN CONDITION BEFORE EXECUTING ***
#   87 K irradiation:  10 uA -> 1.0e-5 A   <- ACTIVE
#    2 K irradiation:   1 uA -> 1.0e-6 A   <- COMMENT IN FOR 2K RUN
d:BeamRef/Current                           = 1.0e-5 A

# Monte Carlo primary-electron budget for statistical precision.
# FIX-4 (2026-06-12): 100,000 histories across a 25x72x50 = 90,000-bin grid
# averages to ~1 history/bin, giving NUI = sigma/mean >> 0.05 in Dose_R and
# Dose_Z. Raised to 10,000,000 primary electrons to bring per-bin statistics
# down to ~1-2% RSD.
#
# FIX-9 splits the 10M primary electrons across three TOPAS beam sources:
#   6M main broad raster + 2M upper-end boost + 2M lower-end boost.
